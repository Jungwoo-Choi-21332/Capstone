멤리스터 웨이퍼 검사 프로젝트 — 쉬운 설명서

이 프로젝트는 뭔가요?

반도체 공장에서 웨이퍼(반도체 원판)에 결함이 생기면 AI가 찾아줘요. 근데 일반 칩은 전기를 많이 먹으니까, 멤리스터라는 저전력 부품으로 AI를 돌리는 거예요.

뭘 했나요?

1. 코드 오류 발견

기존 코드에서 설정값 이름이 틀려서 멤리스터가 실제로는 안 먹히고 있었어요.

에어컨 리모컨에서 "온도" 대신 "은도"라고 써서 에어컨이 무시하고 있던 상황

2. 작동 방식 개선

기존에는 멤리스터를 끼운 채로 학습해서 느리고 비효율적이었는데, 먼저 빠르게 학습하고 → 나중에 멤리스터를 끼워서 테스트하는 방식으로 바꿨어요.

3. 최적 설정 찾기

219가지 조합을 자동 실험해서 최적 설정을 찾았어요.

4. 더 강하게 만들기

  • 학습 횟수 늘림: 30번 → 50번
  • 일부러 노이즈를 넣으며 훈련: 멤리스터가 넣는 잡음에 미리 적응시킴
  • 판정 기준 조정: 0.50 → 0.45로 살짝 낮춰서 결함을 놓치지 않게
